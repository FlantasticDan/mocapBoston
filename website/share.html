What is mocapBoston?

mocapBoston is a motion capture installation in Boston, MA at Emerson College developed by Daniel Flanagan. The installation allows participants to create an animation of their performance by wearing computer-vision enabled stickers which are processed in near-realtime by an algorithm developed for this project which has since been made avaialable open-source.
